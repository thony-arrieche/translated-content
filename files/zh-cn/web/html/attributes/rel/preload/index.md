关键字 **`preload`** 作为元素 {{HTMLElement("link")}} 的属性 [`rel`](/zh-CN/docs/Web/HTML/Element/link#rel) 的值，表示用户十分有可能需要在当前浏览中加载目标资源，所以浏览器必须预先获取和缓存对应资源。

更多细节见[通过 rel="preload"进行内容预加载](/zh-CN/docs/Web/HTML/Preloading_content)。
