{{APIRef("HTML DOM")}}

{{domxref("HTMLVideoElement")}} の **`disablePictureInPicture`** プロパティは HTML の属性を反映し、ユーザーえーえじぇんとがピクチャインピクチャ機能をユーザーに提案するか、自動的にリクエストするかを示します。

## 値

論理値で、`true` ならばユーザーエージェントがこの機能をユーザーに提案すべきです。
