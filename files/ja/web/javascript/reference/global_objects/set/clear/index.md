{{JSRef}}

**`clear()`** メソッドは、 `Set` オブジェクトからすべての要素を取り除きます。

{{EmbedInteractiveExample("pages/js/set-prototype-clear.html")}}

## 構文

```js
clear();
```

### 返値

{{jsxref("undefined")}} です。

## 例

### clear() メソッドの使用

```js
var mySet = new Set();
mySet.add(1);
mySet.add("foo");

mySet.size; // 2
mySet.has("foo"); // true

mySet.clear();

mySet.size; // 0
mySet.has("bar"); // false
```
