A propriedade **`Number.NaN`** representa Not-A-Number (Não-Número). Equivalente a {{jsxref("NaN")}}.

Você não precisa criar um objeto {{jsxref("Number")}} para acessar esta propriedade estática (use `Number.NaN`).
